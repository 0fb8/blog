# 手ごろに試せる Markdown サンプル

> これは Markdown → HTML 変換の動作確認用の短いサンプルです。

---

## 見出しと段落
Markdown は軽量な記法で、テキストをすばやく整形できます。  
改行は行末に半角スペース2つを入れると反映されます。

### 小見出し
説明文をもう少し追加します。太字や斜体も使えます：**太字** と *斜体*。

---

## 箇条書きと番号付きリスト
- **ポイント:** シンプルな記法で読みやすい
- **対応:** 多くのツールや言語でサポート
- **用途:** ドキュメント、メモ、ブログ

1. **手順1:** ファイルを作成 `sample.md`
2. **手順2:** 変換コマンドを実行
3. **手順3:** 出力 HTML をブラウザで確認
